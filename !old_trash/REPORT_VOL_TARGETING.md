## 1. Что сделано

| # | Изменение | Описание |
|---|-----------|----------|
| 1 | **Откат inverse vol** | Вернуты равные веса (1/top_n для long, -1/top_n для short) |
| 2 | **Portfolio vol targeting** | `leverage_scalar = baseline_vol / market_vol`, capped at 1.0 |
| 3 | **Holding** | Сетка [12, 18, 24, 36] баров (убраны 48, 72) |

### Логика vol targeting

```python
BASELINE_VOL = 0.02  # Целевая волатильность рынка
market_vol = np.nanstd(ret[t-lookback:t])  # Текущая vol рынка
leverage_scalar = min(BASELINE_VOL / max(market_vol, 1e-6), 1.0)
pos[t, :] *= leverage_scalar  # Масштабируем все позиции
```

При market_vol > 0.02 → leverage_scalar < 1.0 → позиции уменьшаются
При market_vol < 0.02 → leverage_scalar = 1.0 → полный размер

## 3. Почему vol targeting провалился

### 3.1. BASELINE_VOL = 0.02 слишком низок

Средняя волатильность 4h баров крипты: **0.03-0.05** (3-5% за 4 часа).

| market_vol | leverage_scalar | Эффект |
|------------|----------------|--------|
| 0.02 (тихо) | 1.00 | Полный размер ✅ |
| 0.03 (норма) | 0.67 | **Снижение на 33%** ❌ |
| 0.04 (повышено) | 0.50 | Снижение на 50% ❌ |
| 0.05 (шторм) | 0.40 | Снижение на 60% ❌ |

**Проблема:** система **постоянно** торгует уменьшенным размером (leverage ~0.5-0.67), потому что нормальная vol крипты выше baseline. Сигнал подавлен на 33-50% в **нормальных** условиях.

### 3.2. Сравнение с предыдущими версиями

| Версия | Средний leverage | Результат |
|--------|-----------------|-----------|
| v3 (без vol targeting) | 1.0 (всегда полный) | Sharpe 1.63 |
| v5 (vol targeting) | ~0.5-0.67 (постоянно снижен) | Sharpe 0.18 |

Vol targeting снизил размер позиции **в 1.5-2 раза**, что убило и signal, и PnL.

## 4. Как сделать vol targeting правильно

### 4.1. Использовать относительный baseline

Вместо фиксированного 0.02 использовать **rolling median** волатильности:

```python
# Правильно: текущая vol vs её же медиана
median_vol = np.nanmedian(ret[t-252:t])  # Медиана за ~42 дня
leverage_scalar = min(median_vol / max(market_vol, 1e-6), 1.0)
```

Тогда leverage_scalar ≈ 1.0 в норме и снижается только при аномальном росте vol.

### 4.2. Или: absolute vol targeting

```python
# Целевая vol = 15% годовых → 0.15 / sqrt(6*365) ≈ 0.0033 за бар
TARGET_VOL = 0.0033
leverage_scalar = min(TARGET_VOL / max(market_vol, 1e-6), 2.0)  # max 2x leverage
```
